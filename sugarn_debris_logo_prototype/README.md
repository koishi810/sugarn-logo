# S^n Debris Logo Prototype

Base prototype:

- `sugarn_debris_logo_prototype.pde`: moving debris using large rings and long lines.

Function-based variants:

- `sugarn_debris_polar_wave`: polar wave loops, `r(theta)=base+amp*sin(k theta+t)`
- `sugarn_debris_lissajous`: Lissajous curves
- `sugarn_debris_superellipse`: superellipse outlines
- `sugarn_debris_sine_tracks`: long sine tracks crossing the canvas
- `sugarn_debris_rose_curves`: rose curves

Shared controls:

- `Space`: pause / play
- `H`: hexagon / circle crop
- `R`: reseed debris
- `S`: save screenshot
