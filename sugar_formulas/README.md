# Sugar^n Processing Formula Sketches

Each sketch is independent. Open the `.pde` inside its matching folder in Processing.

Shared controls:

- `Space`: pause / play
- `V`: line / volume
- `S`: save screenshot

The current formula is drawn into the canvas, so screenshots include the mathematical rule.

Sketches:

1. `sugar01_superformula_crystal` - stacked Superformula slices
2. `sugar02_fourier_sugar_threads` - Fourier/Lissajous sugar-thread loops
3. `sugar03_lsystem_sugar_branch` - recursive sugar crystal branching
4. `sugar04_parametric_sugar_shell` - parametric twisted sugar shell
5. `sugar05_voronoi_crystal_cells` - drifting Voronoi crystal cell boundaries
6. `sugar06_reaction_diffusion_frost` - Gray-Scott sugar frost
7. `sugar07_gyroid_sugar_lattice` - gyroid minimal-surface lattice contours
8. `sugar08_phyllotaxis_sugar_core` - golden-angle phyllotaxis sugar core
9. `sugar09_metaball_sugar_mass` - implicit metaball sugar mass
